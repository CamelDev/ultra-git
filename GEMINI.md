# UltraGIT - AI Collaboration Guidelines

This file provides context for the Gemini AI assistant to maintain the project's high bar for performance and aesthetics.

## Core Directives
- **Performance First**: Since UltraGIT is a developer tool, speed is a feature. Use Bun and optimized Git spawning.
- **Premium Aesthetics**: Maintain the "Wow" factor. Every UI element should feel premium, with smooth transitions and consistent dark-mode styling.
- **State Management**: Use Zustand for UI state and active tab isolation to handle multi-repository workflows efficiently.
- **Safety**: Always ensure Git operations are non-destructive and provide clear feedback/undo options.
